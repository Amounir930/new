## 📐 1. القرارات المعمارية الكبرى (15 قرار)

| # | القرار | الاختيار | السبب |
|---|--------|----------|-------|
| 1 | المبالغ المالية | `BIGINT` (سنت/هللة) | Stripe Standard — لا أخطاء تقريب، أسرع 10× |
| 2 | المعرّفات | ULID مُخزّن كـ `UUID` (16 bytes) | Sortable + cursor pagination |
| 3 | محاذاة الأعمدة | Fixed أولاً → Variable آخراً | توفير RAM padding |
| 4 | العلامات (Tags) | `TEXT[]` (PostgreSQL Array) | أسرع من JSONB للبحث المباشر |
| 5 | تعدد اللغات | `JSONB` → `{"ar":"..","en":"..","fr":".."}` | مرونة لا نهائية بدون تعديل Schema |
| 6 | الحقول المشفرة S7 | `TEXT` أو `BYTEA` (بلا VARCHAR) | طول ناتج التشفير متغير |
| 7 | حالات الطلب/الشحن | Native `PG ENUM` | 4 bytes فقط + فلترة سريعة |
| 8 | الأوزان | `INTEGER` بالجرامات فقط | التحويل للكيلو في الواجهة |
| 9 | الحذف | Soft Delete → `deleted_at TIMESTAMPTZ` | لا حذف فعلي أبداً (محاسبة + SaaS) |
| 10 | FK المحاسبية | `RESTRICT` (لا CASCADE) | حماية الإيصالات والسجلات المالية |
| 11 | استيراد جماعي | `DEFERRABLE INITIALLY DEFERRED` FKs | 5× أسرع |
| 12 | Vector Search | `HNSW` index (لا IVFFlat) | 20× أسرع + لا rebuild |
| 13 | Count | Denormalized counters | حظر `COUNT(*)` على الجداول الكبيرة |
| 14 | API Pagination | Cursor-based (ULID) فقط | حظر OFFSET/LIMIT |
| 15 | الأسعار + العملات | Composite Type `money_amount` | لا يفترق الرقم عن عملته أبداً |

### 2.10 Pre-Aggregation Views

```sql
-- Best Sellers (refresh كل ساعة)
CREATE MATERIALIZED VIEW mv_best_sellers AS
SELECT p.id, p.name, p.slug, p.base_price, p.main_image,
       COALESCE(SUM(oi.quantity), 0) AS total_sold
FROM products p
LEFT JOIN order_items oi ON p.id = oi.product_id
LEFT JOIN orders o ON oi.order_id = o.id
  AND o.status IN ('delivered','shipped') AND o.deleted_at IS NULL
WHERE p.is_active = true AND p.deleted_at IS NULL
GROUP BY p.id ORDER BY total_sold DESC;

-- Billing Metrics (refresh كل ساعة)
CREATE MATERIALIZED VIEW mv_tenant_billing AS
SELECT tenant_id, date_trunc('month', created_at) AS period,
       COUNT(*) AS orders_count, SUM(total) AS revenue
FROM orders WHERE deleted_at IS NULL
GROUP BY 1, 2;
```
